# figma2assets

В каталоге собраны два независимых подхода для подготовки ассетов из Figma:

## figma-plugin

Development-плагин Figma (экспорт PNG верхних детей, `manifest.json`, atlas, TS-файлы).
Работает без Figma REST API, поэтому это основной и рекомендуемый путь.

Подключается через приложение Figma, не через браузер!

## figma-api

Node-скрипт `figma2phaser` для генерации через Figma REST API (`FIGMA_TOKEN`/`FIGMA_API_TOKEN`).
Нужен, если требуется автоматизация вне интерфейса Figma.

Важно:

- На бесплатном тарифе Figma API лимиты очень жесткие (на практике часто это всего несколько запросов в месяц для нужных
  endpoint), поэтому для стабильной работы рекомендуется использовать `figma-plugin/`.
